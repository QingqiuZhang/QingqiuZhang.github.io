Iris Dataset Analytics

Author: Qingqiu Zhang Date: 2022-03-28 Category: Data Visualization Python Classification Machine Learning

The Iris Data set was created by R.A. Fisher and is perhaps the best known data set to be found in the pattern recognition literature. Fisherâ€™s paper is a classic in the field and is referenced frequently to this day. The data set contains 3 classes of 50 instances each, where each class refers to a type of iris plant. Predicted attribute: class of iris plant.

This article aims to classify this dataset utilizing the following models: K Nearest Neighbors (KNN), Naive Bayes and Logistic Regression.

Train and evaluate three models using cross-validation

K-Nearest Neighbors Classifier

import sklearn.neighbors
import sklearn.metrics
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler

num = 200
results = []
features = ['sepal length', 'sepal width', 'petal length', 'petal width']
for k in range(1, 20):
    test_accuarcy_per_time = []
    for i in range(num):
        # Split data
        train_data, test_data = train_test_split(df, test_size=0.25)
        # Standardization 
        scaler = StandardScaler()
        scaler.fit(train_data.iloc[:, 0:4])
        df_normalized = pd.concat([pd.DataFrame(scaler.transform(df.iloc[:, 0:4]), 
                                                columns=features), 
                                   df[['class']]], 
                                  axis=1)
        train_normalized = df_normalized.iloc[train_data.index]
        test_normalized = df_normalized.iloc[test_data.index]
        train_normalized_x = train_normalized[features].values
        train_normalized_y = train_normalized['class'].values
        test_normalized_x = test_normalized[features].values
        test_normalized_y = test_normalized['class'].values
        # Perform KNN
        knn = sklearn.neighbors.KNeighborsClassifier(n_neighbors=k).fit(train_normalized_x,
                                                                        train_normalized_y)
        # Store the test accuracy of each split into test_accuarcy_per_time
        # Store the test accuracy of each split into test_accuarcy_per_time
        test_accuarcy_per_time.append(sklearn.metrics.accuracy_score(test_normalized_y,
                                                                     knn.predict(test_normalized_x)))
    # Calculate the average test accuracies for different ks and store them into results
    results.append({'k':k, 'test accuracy':np.mean(test_accuarcy_per_time)})
results = pd.DataFrame(results)
print(results)

Naive Bayes Classifier

from sklearn.naive_bayes import MultinomialNB

features = ['sepal length', 'sepal width', 'petal length', 'petal width']
df1 = df.copy()
for column in features:
    df1[column] = [round(i - np.min(df1[column])) for i in df1[column]]
    df1[column] = df1[column].astype('category')
X = pd.get_dummies(df1[features])

# Calculate mean test accuracy
test_accuarcy_per_time = []
for i in range(num):
    X_train, X_test, y_train, y_test = train_test_split(X, df1['class'].values, test_size=.25)
    mnb = MultinomialNB(alpha=0.01)
    mnb.fit(X_train, y_train)
    y_test_pred = mnb.predict(X_test)
    test_accuarcy_per_time.append(sklearn.metrics.accuracy_score(y_test, y_test_pred))
print('\n',np.mean(test_accuarcy_per_time))

Logistic Regression

from sklearn.linear_model import LogisticRegression

df2 = df.copy()
X = df2.iloc[:, 0:4].values
y = df2.iloc[:, 4].values

# Calculate mean test accuracy
test_accuarcy_per_time = []
for i in range(num):
    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size = 0.25)
    sc = StandardScaler()
    X_train = sc.fit_transform(X_train)
    X_test = sc.transform(X_test)
    lr = LogisticRegression(penalty="l2", solver='lbfgs', multi_class='auto')
    lr.fit(X_train, y_train)
    y_pred = lr.predict(X_test)
    test_accuarcy_per_time.append(sklearn.metrics.accuracy_score(y_test, y_pred))

# accuracy score

Conclusion

For the current running, Logistic Regression has the highest test accuracy followed by KNN Classifier. Naive Bayes Classifier has the lowest test accuracy.

However, when I tried more times, I found that sometimes Logistic Regression has the highest test accuracy while other times KNN Classifier does. Naive Bayes Classifier always has the lowest test accuracy.
